# PLACEMENT QUEST - CONTENT CREATION GUIDE

**Editorial Standards, Tone, and Content Strategy for Private Equity Placement Agent Intelligence**

## üéØ Content Mission

Create the most **authoritative, accurate, and actionable** placement agent intelligence that helps fund managers, LPs, and placement agents make informed decisions about fundraising, agent selection, and market positioning.

## üìä Content Quality Standards

### E-E-A-T Optimization (Experience, Expertise, Authoritativeness, Trustworthiness)
- **Experience**: Real-world fundraising insights, deal examples, case studies
- **Expertise**: SEC filings, Form ADVs, official fund documents, verified AUM data
- **Authoritativeness**: Link to SEC EDGAR, Preqin, PEI, official placement agent sites
- **Trustworthiness**: Transparent about data sources, clear disclaimers, updated quarterly

### Citation Requirements
- **SEC Sources**: Always link to EDGAR filings, Form ADVs
- **Financial Claims**: Cite official fund documents, regulatory filings
- **AUM Data**: Reference Preqin, PEI Data, firm announcements
- **Fee Structures**: Use industry reports, verified client testimonials
- **Deal Information**: Link to press releases, LP announcements

## ‚úçÔ∏è Voice & Tone

### Professional but Accessible
- **Expert without jargon**: Explain PE terms clearly (IRR, DPI, TVPI, etc.)
- **Data-driven**: Back all claims with verifiable sources
- **Direct**: No fluff, focus on actionable intelligence

### Writing Style
- **Concise**: Busy professionals need quick insights
- **Structured**: Use tables, bullet points, comparison matrices
- **Action-oriented**: Help readers make placement agent decisions

### Example Tone:
‚ùå **Avoid**: "This amazing placement agent is the best in the business!"
‚úÖ **Use**: "Campbell Lutyens raised $45B across 150+ funds (2020-2024), ranking #1 by capital raised according to PEI's 2024 Placement Agent League Tables."

## üéØ Content Quality Scoring System (0-100)

### Scoring Algorithm
Content quality is measured on a 0-100 scale based on:

**Content Depth (40 points max)**
- 3000+ words: 40 points
- 2000+ words: 35 points  
- 1500+ words: 30 points
- 1000+ words: 20 points
- 500+ words: 10 points

**Data Quality (25 points max)**
- Verified AUM/deal data: +15 points
- 10+ authoritative sources: +10 points
- 5+ authoritative sources: +7 points
- 2+ authoritative sources: +3 points

**Financial Intelligence (20 points max)**
- Fee structure analysis: +10 points
- Performance metrics: +5 points
- Competitive positioning: +5 points

**Currency/Freshness (15 points max)**
- Less than 1 month old: +15 points
- Less than 3 months old: +10 points
- Less than 6 months old: +5 points

### Content Enhancement Priority
**High Priority (Score < 60):**
- Add verified AUM and deal data
- Include fee structure comparisons
- Expand to 2000+ words
- Add 10+ authoritative sources

**Medium Priority (Score 60-79):**
- Update with latest fund closes
- Add competitive analysis
- Improve data visualizations

**Low Priority (Score 80+):**
- Monthly freshness updates
- Monitor for new deals
- Expand based on market changes
